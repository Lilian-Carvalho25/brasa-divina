# Brasa Divina 🍕
Bem vindo(a)! Este site está sendo desenvolvido com o objetivo de representar um site real para uma pizzaria. Nasceu como um Projeto TCC da Faculdade. <br>

## ⚙ Funcionalidades 
- Páginas sêmanticas e bem construídas <br> 
- Layout lindo e bem elaborado <br> 
- Design responsivo <br>
- Registro das informações informadas pelos usuários, no banco de dados. ⚙️ <br><br>
